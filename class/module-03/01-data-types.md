### 1️⃣ Integer (`int`)

**Definition:**
An integer is a **whole number** (no decimal).

**Example:**

```python
age = 25
print(age)
print(type(age))
```

---

### 2️⃣ Float (`float`)

**Definition:**
A float is a **number with decimal points**.

**Example:**

```python
price = 99.99
print(price)
print(type(price))
```

---

### 3️⃣ Boolean (`bool`)

**Definition:**
Boolean has only **two values**: `True` or `False`.

**Example:**

```python
is_student = True
print(is_student)
print(type(is_student))
```

---

### 4️⃣ String (`str`)

**Definition:**
A string is **text or characters** written inside quotes.

**Example:**

```python
name = "Rahim"
print(name)
print(type(name))
```

---

### 5️⃣ Complex (`complex`)

**Definition:**
Complex numbers have **real and imaginary parts**.

**Example:**

```python
number = 3 + 5j
print(number)
print(type(number))
```

---

### 6️⃣ List (`list`)

**Definition:**
A list stores **multiple values** and can be **changed (mutable)**.

**Example:**

```python
fruits = ["apple", "banana", "mango"]
print(fruits)
print(type(fruits))
```

---

### 7️⃣ Tuple (`tuple`)

**Definition:**
A tuple is like a list but **cannot be changed (immutable)**.

**Example:**

```python
colors = ("red", "green", "blue")
print(colors)
print(type(colors))
```

---

### 8️⃣ Set (`set`)

**Definition:**
A set stores **unique values only** (no duplicates).

**Example:**

```python
numbers = {1, 2, 3, 3, 4}
print(numbers)
print(type(numbers))
```

---

### 9️⃣ Dictionary (`dict`)

**Definition:**
A dictionary stores data in **key : value pairs**.

**Example:**

```python
student = {
    "name": "Rahim",
    "age": 20,
    "course": "Python"
}
print(student)
print(type(student))
```
